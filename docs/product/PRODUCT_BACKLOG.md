# Product Principles

Journey Builder should become:

- Enterprise-first
- Low cognitive load
- Highly scalable
- Accessible
- Designer-friendly
- Developer-friendly
- Configuration over code
- Predictable
- Fast
- Visually calm

Every future enhancement should strengthen these principles rather than increase complexity.
